## What I Build

### BOWNH — AI Agent Orchestration Platform
An AI agent orchestration platform I built from zero to MVP, independently.

Two agents shipped and in production:

- **AI Market Research Agent** — synthesises public data sources into structured market sizing, competitor positioning, user-pain signals, and strategic gaps. Conversational interface built for non-technical users.
- **AI Risk Assessment Agent** — takes a financial profile and returns a structured credit risk verdict with recommended clauses and mitigation options.

A third concept is in design: an on-device PRD agent built privacy-first for data-sensitive enterprises, addressing the data-residency constraints that block cloud AI adoption at scale.

I authored the PRD, designed the agent architecture, made the prioritisation calls, and shipped end-to-end.
